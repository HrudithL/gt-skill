# Examples

Concrete reference tables. Each shows one archetype with its source code
and rendered output. Load the matching `.py` for the archetype you need.

| Archetype | Use when... | Files |
|---|---|---|
| Financial | Money, prices, signed deltas, percentages | `financial.py` · `financial.png` |
| Time series | Dates, trends, monthly/yearly aggregation | `time_series.py` · `time_series.png` |
| Ranking | Top-N lists, ordered results | `ranking.py` · `ranking.png` |
| Summary stats | Aggregations, totals, subtotals | `summary_stats.py` · `summary_stats.png` |
| Scientific | Measurements with units, sig figs | `scientific.py` · `scientific.png` |
| Heatmap | Color-encoded data cells | `heatmap.py` · `heatmap.png` |
