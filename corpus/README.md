# Corpus

Hand-curated reference tables for evaluating great_tables output. Each
subdirectory is one archetype and ships a `good_table` / `bad_table`
contrast plus a `rationale.md` explaining the design choices. The judge
(issue #6) and the skill improvement agent (issue #7) read this corpus
as their ground-truth reference. Authoring rules and acceptance
criteria live in `.planning/01-corpus.md`.
